Инструкция

1. Подготовьте файл

  • Формат файла: Excel (.xlsx / .xls)
  • В файле должны быть оценки по 4 показателям от 0 до 10

2. Проверьте корректность данных

  • Если кто-то другой закрыл обучение за вас, сначала исправьте направление вручную
  • Если есть сомнения по направлению, скорректируйте файл перед загрузкой
  • Проверка делается раз в квартал и занимает несколько минут

3. Загрузите файл

  • Нажмите «Выберите файл»
  • Выберите Excel
  • Нажмите «Рассчитать»

4. Посмотрите результат

  • На первой странице — сводка по направлениям
  • На второй странице — сравнительный график
  • Жёлтая линия на графике показывает порог 80%

5. Скачайте итоговый файл

  • Нажмите «Скачать Excel»

Безопасность

  • Данные никуда не отправляются
  • Обработка идёт локально в браузере
